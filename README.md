# Virtuis - Plateforme d'Investissement Moderne

Virtuis est une plateforme d'investissement moderne, responsive et sécurisée construite avec React, Next.js, Tailwind CSS et Firebase.

## 🚀 Fonctionnalités

### Pour les utilisateurs
- ✅ Authentification sécurisée avec Firebase
- ✅ Tableau de bord utilisateur complet
- ✅ Plans d'investissement flexibles
- ✅ Dépôts via MTN Mobile Money et Orange Money
- ✅ Retraits sécurisés
- ✅ Système de parrainage avec bonus
- ✅ Historique des transactions
- ✅ Notifications en temps réel
- ✅ Gestion de profil

### Pour les administrateurs
- 📊 Tableau de bord avec statistiques complètes
- 👥 Gestion des utilisateurs
- ✅ Validation des dépôts et retraits
- 💼 Gestion des plans d'investissement
- 🎁 Gestion des bonus de parrainage
- 📢 Envoi de notifications
- 🚫 Blocage/déblocage d'utilisateurs
- 🔍 Recherche avancée

## 🛠️ Stack Technologique

- **Frontend**: React 18, Next.js 14
- **Styling**: Tailwind CSS
- **Backend**: Firebase (Auth, Firestore, Storage)
- **État**: Zustand
- **Notifications**: React Hot Toast

## 📋 Prérequis

- Node.js 18+
- npm ou yarn
- Compte Firebase

## 🔧 Installation

```bash
git clone https://github.com/romeopedezutso-wq/virtuis.git
cd virtuis
npm install
cp .env.example .env.local
npm run dev
```

Accès : `http://localhost:3000`
